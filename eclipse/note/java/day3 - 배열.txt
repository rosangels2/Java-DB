배열
 - 같은 의미와 같은 자료형을 가지는 변수들의 집합
 - 0번지부터 배열의 크기 -1번지까지 접근이 가능
 - 배열 선언
자료형[] 배열이름1,배열이름2;
자료형 배열이름[],배열이름2; []이 배열 이름 뒤에 붙으면 , 뒤에 변수들은 배열이 아닌 일반 변수로 선언된다 


ex)
int num1 = 8, num2 = 12;
int gcd = 1;
int i = 1;
int arr[] = new int[4]
0번지:첫번째 정수, 1번지:두번째 정수, 2번지:최대공약수, 3번지:반복문 횟수
arr[0] = 8;
arr[1] = 12;
arr[2] = 1;
arr[3] = 1;
//같은 정수여도 같은 의미가 아니기 때문에 배열을 만드는 의미가 없음
