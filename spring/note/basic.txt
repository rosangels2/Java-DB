URI ->> 컨트롤러 -> jsp -> DAO -> vo클래스 -> DB

URI - 컨틀롤러에게 기능 요청
컨트롤러 - 필요한 작업에 맞는 메서드를 서비스에 호출
jsp - Vo클래스를 통해 DB에서 데이터를 수신
DAO - DB에 접근할 수 있는 메서드의 집합체
vo(value object)클래스 - DB에서 데이터 집합을 한번에 가져오기 위한 클래스
DB - 필요한 데이터들을 모아놓은 데이터베이스


폴더 구조

src > main > webapp > WEB-INF > spring > appServle > servlet-context.xml
				         root-context.xml
			         views > home.jsp
			         web.xml
